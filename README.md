# Inkblot

Augmented Reality for Romance Readers — experience indie romance like never before.

## About

Inkblot is a subscription service delivering curated indie romance books with scents, objects, playlists, and secret scenes — a velvet, slow-burn reading ritual.

## Tech Stack

- **Vite** — Fast build tool
- **React 18** — UI framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling
- **shadcn/ui** — Component library
- **React Router** — Navigation

## Getting Started

```sh
# Clone the repository
git clone https://github.com/ggavr/inkblot.git

# Navigate to project
cd inkblot

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:8080`

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run preview` — Preview production build
- `npm run lint` — Run ESLint
